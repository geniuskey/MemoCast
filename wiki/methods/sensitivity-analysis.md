# Sensitivity Analysis

Sensitivity analysis는 특정 입력 변수를 변화시켰을 때 결과가 얼마나 바뀌는지 보는 방법입니다.

## 예시

스마트폰 출하량을 -20%, -10%, 0%, +10%, +20%로 바꿔 DRAM/NAND 수요 변화를 계산할 수 있습니다.

## MemoCast에서의 용도

- 수요 driver의 중요도 비교
- bull/base/bear case의 차이 설명
- 입력 가정 오류가 결과에 미치는 영향 확인
