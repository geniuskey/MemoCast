# Memory Supply Chain and Equipment Constraints

Equipment/fab/material 자료는 capex가 곧바로 bit supply로 바뀌지 않는 이유를 설명한다. 300mm fab 투자, advanced packaging, test 장비, HBM wafer allocation, 지역/수출통제 제약을 분리해서 봐야 한다.

## 핵심 해석

- SEMI/KLA/Advantest/Teradyne류 자료는 memory upcycle이 front-end wafer뿐 아니라 packaging/test 장비 병목을 동반함을 보여준다.
- China localization/CXMT/YMTC/수출통제 자료는 장기 supply ceiling과 regional substitution risk를 동시에 만든다.
- 물/전력/material 제약은 fab capacity ramp의 non-financial bottleneck이다.

## 정리: 공급 제약을 demand forecast에 연결하는 법

이 페이지의 핵심 정리 결과는 “장비 투자가 늘었다”가 아니라, 어떤 병목이 어떤 제품군의 **실현 가능한 bit supply**를 늦추는지다. HBM은 wafer input뿐 아니라 TSV/advanced packaging/test/KGD 검증을 통과해야 하므로, DRAM/NAND 일반 capex보다 더 많은 gate를 가진다. ^[raw/datasets/dram-wafer-capacity-allocation-2026.md] ^[raw/reports/equip-advantest-hbm-test-lead-times-2026.md]

| Bottleneck | Directly constrains | Demand forecast consequence | Watch signal |
| --- | --- | --- | --- |
| HBM wafer multiple | conventional DRAM wafer availability | HBM demand가 conventional DRAM shortage를 악화시킬 수 있다 | HBM output share vs revenue share, DDR5/server allocation |
| Advanced packaging / CoWoS / hybrid bonding | high-end GPU/HBM module realization | announced accelerator demand를 installed HBM EB로 할인 | CoWoS allocation, TCB/hybrid bonding orders |
| Memory test / burn-in / KGD | HBM4 and high-density DRAM ramp | roadmap spec를 즉시 volume shipment로 보지 않는다 | tester lead time, memory test revenue mix |
| 300mm fab capacity lag | broad DRAM/NAND bit supply | capex 증가는 12~24개월 후 supply curve로 반영 | 300mm capacity growth, WFE shipment timing |
| Power/water/material/site constraints | regional fab expansion and data-center deployment | supply와 demand 양쪽 realization gate로 적용 | grid interconnect, water/energy commitments |
| Export control/localization | regional substitution and stranded capacity | China/non-China supply ceiling을 분리 | KLA/ASML China exposure, CXMT/YMTC ramp |

### Capacity-to-bit conversion rule

MemoCast에서 capex headline을 만났을 때는 아래 순서로만 bit supply에 반영한다.

1. **Capex type 분리**: wafer fab, packaging, test, power/site, R&D 중 무엇인지 먼저 분류한다.
2. **Lag 부여**: WFE와 fab 증설은 12~24개월, packaging/test는 병목 완화 시점을 별도 적용한다.
3. **Product allocation 적용**: HBM, server DRAM, mobile/PC DRAM, NAND가 같은 wafer/capex를 동일하게 나눠 갖지 않는다.
4. **Yield/ramp discount**: HBM4, 3D DRAM, high-layer NAND처럼 신공정은 초기 yield와 qualification gate를 둔다.
5. **Realization factor 산출**: 가장 낮은 gate를 supply ceiling으로 둔 뒤 [[channel-pricing-signal]]의 shortage regime과 교차 확인한다.

### 예측 모델 연결 규칙

1. `constraintScore ≥ 75`: headline capex보다 **realization factor**를 먼저 적용한다. HBM4처럼 packaging/test 의존도가 높은 제품은 demand EB가 있어도 출하 EB로 전량 실현되지 않는다.
2. `55 ≤ constraintScore < 75`: 일부 공급축이 병목이다. wafer input, CoWoS/TCB, tester capacity 중 가장 낮은 gate를 supply ceiling으로 둔다.
3. `35 ≤ constraintScore < 55`: 장비 투자는 늘지만 12~24개월 전환 lag가 남아 있다. 단기 가격/shortage 모델에는 공급 완화를 지연 반영한다.
4. `constraintScore < 35`: expansion 상태다. bit supply 모델에서 capex와 wafer capacity 증가를 비교적 직접 반영하되, 제품별 wafer multiple은 유지한다.
