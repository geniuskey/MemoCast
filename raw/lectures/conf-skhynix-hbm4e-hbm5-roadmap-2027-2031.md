# SK hynix HBM4E/HBM5 로드맵 (2027-2031) — extracted notes

> 차세대 HBM 용량·대역폭 로드맵이 가속기당 HBM 콘텐츠 상한 → 장기 HBM bit 수요 곡선.

## HBM4E (2026 샘플 → 2027 양산)

- **32Gb density**(HBM4 대비 +33%) → **12-Hi로 48GB**(HBM4는 16-Hi로 48GB).
- pin speed 최대 **16Gbps**(HBM4 대비 +37%) → stack당 **4 TB/s**(기록).
- custom HBM4E: HBM 컨트롤러를 base die로 이동(GPU/ASIC compute 면적·인터페이스 전력 개선). TSMC 협력.
- 2026-2028: HBM4 16-Hi, HBM4E 8/12/16-Hi + custom.

## HBM5/HBM5E (2029-2031)

- HBM5/HBM5E + custom HBM5, GDDR7-Next, DDR6, 400+ layer 4D NAND 로드맵.
- (별도 파일) HBM5 4096-bit I/O, on-stack L3 cache 전망.

## CES 2026

- SK hynix 16-layer HBM4 기록.

## 메모리 수요 예측 관련성

- 세대당 capacity(48GB→64GB+)·대역폭(4TB/s→) 상승 = 가속기당 HBM GB 증가 → bit 수요 상방.
- HBM4E 12-Hi 48GB는 동일 용량을 더 낮은 stack으로 → yield/공급 개선(공급 시나리오).
- 로드맵 타임라인(HBM4E 2027, HBM5 2029)이 장기 HBM 수요 곡선의 변곡점.
