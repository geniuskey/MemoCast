# 양자/극저온(Cryo-CMOS) 메모리 — extracted notes

> 장기 niche. 단기 메모리 수요 영향은 미미하나 새로운 메모리 카테고리로 기록.

## 핵심 개념

- Cryo-CMOS: 4K~mK에서 동작하는 CMOS IC. 양자 프로세서-상온 전자장치 간 wiring 병목 해결, 수백 큐빗 이상 확장 enabler.
- 큐빗 제어 프로세서·메모리 블록이 극저온에서 동작 필요. cooling power 예산은 mK 단계 microwatt 수준.

## 주요 사실/수치

- 4.2K 환경 40nm bulk CMOS에서 채널 온도 상승 50K+ 측정(열설계 가정 무효화) → 신뢰성 제약.
- 4K용 in-memory computing이 cryogenic von-Neumann의 메모리 대역폭 병목 타겟.
- IBM 2026 Kookaburra: logical qubit + quantum memory 첫 통합 시연.

## 메모리 수요 예측 관련성

- 양자컴퓨팅 메모리는 2026 시점 절대 물량 미미 → 단기 bit 수요 영향 거의 없음(long-tail).
- 다만 cryo-CMOS/극저온 메모리는 새로운 메모리 device 카테고리로 장기 모니터링 대상.
- 양자 본격화 시 대량 classical 제어 메모리 수반 가능(장기 시나리오).
