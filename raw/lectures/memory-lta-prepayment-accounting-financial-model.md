# 메모리 LTA 선지급·이연수익 회계 — extracted notes

> 2026 메모리 LTA의 prepayment가 재무제표·수요 가시성에 미치는 영향 이해.

## 회계 개념

- **deferred revenue(이연/선수수익)**: 고객이 선지급한 금액은 제품 인도 전까지 **부채(liability)**로 인식, 인도 시 매출 전환.
- 장기계약 선수금은 **non-current liability**(다년), 단기분은 current liability.

## 메모리 LTA 적용 맥락

- 2026: Microsoft·Google 등이 HBM/DRAM 확보 위해 **5년 LTA + 10~30% 선지급**(prepayment) 체결.
- 공급사 입장: 선수금은 부채로 계상 후 인도분만큼 매출 인식 → 매출 인식 시점/현금흐름 분리.
- 공급사 현금흐름 개선(선수금)·capex 자금 조달 수단. 고객 입장: capacity·가격 floor 확보.

## 메모리 수요 예측 관련성

- 공급사 재무제표의 **deferred revenue/customer prepayment 증가**는 미래 인도 약정(=수요 가시성)의 선행지표.
- 선지급 규모·기간은 LTA 커버리지(가격 floor 적용 물량)의 정량 신호.
- 재무모델링 시 매출 인식 시차(선수금→매출)를 반영해야 분기 매출 예측 정확.
