# 메모리 수요 계절성·분기 패턴 — extracted notes

> 분기 예측에는 계절성 보정이 필요. 단 2026 사이클은 AI가 계절성을 압도.

## 전통적 계절성

- 모바일: 상반기(특히 1Q~2Q) 비수기, 하반기(3Q~4Q) 신제품·홀리데이로 성수기 경향.
- PC: 하반기 back-to-school·홀리데이 성수기, 1Q 비수기.
- 메모리 가격은 수요 계절성 + 재고 사이클로 분기 변동.

## 2026 특이점 (계절성 무력화)

- AI server 수요가 계절성 압도: 1Q26 server DRAM +60% QoQ, conventional DRAM +55~60% QoQ(비수기임에도 급등).
- 4Q25 메모리 가격 +50% QoQ, DRAM 4Q contract +75% YoY.
- 2Q26부터 스마트폰 output adjustment 진입. 모바일 비수기 + 메모리 부족 동시 작용.
- 가격 급등으로 스마트폰 -2% YoY, 노트북 -2.4%(계절 수요 위축).

## 메모리 수요 예측 관련성

- 통상 분기 모델은 계절 더미(seasonal dummy) 필요하나, AI supercycle기에는 계절성 < 구조적 수요.
- 계절성은 consumer(모바일/PC) 세그먼트에 잔존, server/AI 세그먼트는 약함 → 세그먼트별 계절 보정 차등.
- turning point 후 계절성 복원 가능 → 시나리오별 계절 가정 전환.
