# 메모리 분기 계약가 협상 프로세스 — extracted notes

> 분기 contract price가 어떻게 정해지는지(협상 주체·타이밍·전략). 가격 모델의 제도적 기초.

## 협상 프로세스

- 북미·중국 주요 CSP·서버 OEM이 공급사와 **연간 LTA + 분기 가격** 협상.
- **공급사가 먼저 quote 제시**: Samsung·Micron 선제, SK hynix·CXMT는 tentative → 최종 협상 5월 마무리(2Q26).
- 공급사는 server DRAM 우선(고마진) + **catch-up pricing**(응용 간 가격격차 축소)으로 mobile 등 후속 인상.
- CSP가 가격 인상 수용 → 타 고객도 allocation 확보 위해 추종 → 공급사 가격결정력 강화.

## 가격 형성 동학

- 1Q26 DRAM 산업 +81% QoQ(급격 contract surge). 전 제품군 catch-up.
- multi-tiered pricing(세그먼트별 차등)으로 마진 극대화.
- spot(선행)→contract(분기 확정)→실적(후행) 흐름(spot/contract 파일 연계).

## 메모리 수요 예측 관련성

- 분기 contract 협상 타이밍(분기 초 quote, 중순 확정)이 가격 시계열의 step 구조.
- 공급사 선제 quote + CSP 수용은 가격결정력(과점)의 제도적 표현.
- catch-up pricing은 세그먼트 간 가격 수렴 → mix별 ASP 예측 보정.
