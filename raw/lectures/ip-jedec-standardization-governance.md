# JEDEC 표준화 조직·거버넌스 — extracted notes

> 표준이 메모리 상호운용성·세대전환 타이밍·commoditization을 결정 → 수요 구조의 제도적 기반.

## 핵심 개념

- JEDEC Solid State Technology Association: 1944 설립(JETEC), 1958 반도체 확장. 마이크로일렉트로닉스 open standard 개발.
- **360+ 회원사, 3,000+ 자원봉사자, 50+ 위원회**. Board of Directors(회원사 선출) 거버넌스.
- 메인 메모리 표준은 **JC-42 위원회**. 표준 무료 공개(세계 최초 전면 개방).

## 주요 사실

- 메모리 표준 3대 범주: main memory(DDR), flash, mobile(LPDDR).
- 표준은 제조사-구매자 간 오해 제거, 상호교환성·개선 촉진. commoditize로 가격↓·품질 유지 → 혁신 경쟁 유도.

## 메모리 수요 예측 관련성

- DDR/LPDDR/HBM 표준 확정 시점이 세대전환(수요 mix shift) 타이밍을 게이트(별도 conf-/ip- 파일과 연결).
- 표준 commoditization은 가격 경쟁 구조 → 가격 시나리오의 제도적 배경.
- 표준 로드맵(DDR6/HBM5)이 미래 대역폭/용량 상한 → 수요 한계 설정.
