# UFS 5.0 / 4.1 모바일 스토리지 로드맵 — extracted notes

> 모바일 NAND 인터페이스·용량 로드맵이 스마트폰 NAND 콘텐츠 성장 좌우.

## 핵심 사실/수치

- **UFS 5.0**: UFS 4.0 대비 성능 2배, **최대 10.8 GB/s**(MIPI M-PHY 6.0, UniPro 3.0). UFS 4.1 대비도 2배.
- 상용 폰 탑재 **~2027(갤럭시 S27 추정)**. Kioxia가 2026-02 말 UFS 5.0 호환 모듈 샘플 출하.
- 용량: Samsung·Kioxia UFS 5.0 모듈 **512GB·1TB** 제공(평가 플랫폼).
- 4.1이 4.0→5.0 전환 bridge 역할.

## 메모리 수요 예측 관련성

- 인터페이스 세대전환은 디바이스당 NAND 콘텐츠·성능 요구 상승 → 모바일 NAND bit 수요 견인.
- 1TB 모듈 보편화 시 스마트폰 평균 NAND 콘텐츠 상방(특히 on-device AI 캐시).
- 전환 타이밍(2027)이 모바일 NAND 수요 곡선의 변곡점.
