# TrendForce 1Q26 Memory Pricing Cross-check — extracted notes

> 원천 요약 노트. 공급사 실적 ASP/bit 데이터를 한 곳에 교차정리. 수치는 wiki에서 교차검증 후 인용.

## 핵심 수치 (1Q26 ASP / bit 출하)
- Samsung 메모리 ASP: 2025 연평균 대비 +146%; 모바일 메모리 조달원가 YoY +107%
- SK hynix DRAM ASP: QoQ mid-60% 상승, DRAM bit 출하 QoQ 대체로 flat
- SK hynix NAND ASP: QoQ mid-70% 상승, NAND bit 출하 QoQ 약 -10%
- SK hynix R&D: 2.55조원, YoY +68.3%
- (Micron은 본 기사에서 가격/출하 수치 미제시, "key supplier"로만 언급)

## 수요/가이던스 코멘트
- 1Q26은 전형적 비수기임에도 메모리 가격이 전 제품군 급등 → "메모리 슈퍼사이클"
- bit 출하는 flat~감소인데 ASP만 급등 = 가격 주도(price-led) 사이클의 정의적 특징
- HBM4E 샘플 2H26, 양산 2027 (공통 로드맵)

## 메모리 수요 예측 관련성
- bit 정체 + ASP 급등 패턴은 수요량(units) 증가가 아닌 공급 절제/가격 회복이 매출의 핵심임을 보여줌
- 수요예측 모델링 시: 매출 = bit × ASP에서 1Q26은 ASP 기여가 압도적 → bit 수요와 매출을 분리 해석 필요
- DRAM mid-60% vs NAND mid-70% → NAND 가격 회복 모멘텀이 DRAM보다 가파름
