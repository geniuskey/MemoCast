# Rambus 메모리 IP·로열티 (2026) — extracted notes

> 메모리 IP(컨트롤러·PHY·특허)의 로열티 모델. 메모리 수요의 IP 단 동행 지표.

## 핵심 수치

- Rambus 1Q26: 제품매출 **$88.0M(+15% YoY)**, GAAP 매출 $180.2M, 라이선싱 billing $70.8M.
- 업계 최速 **HBM4E 메모리 컨트롤러 IP**. Micron 특허 라이선스 2029년까지 연장(예측가능 로열티).
- Samsung Foundry와 HBM4 모듈 협업. **DDR5 RCD** 리더십이 제품 성장 견인.
- MRDIMM 시장 2026 $600M. HBM4+양자안전 보안 = 2030 $20B 기회.

## 메모리 수요 예측 관련성

- Rambus 제품/IP 매출(DDR5 RCD, HBM 컨트롤러)은 메모리 모듈·HBM 수요의 IP 단 동행 지표.
- HBM4E 컨트롤러·custom base die(별도 ip- 파일)는 HBM 세대전환 ramp의 IP 의존성.
- 특허 로열티(Micron 등)는 메모리 출하량과 연동 → 안정적 수요 프록시.
