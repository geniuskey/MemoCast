# 스마트폰 메모리 콘텐츠·spec 다운그레이드 (2026)

> 가격 급등으로 콘텐츠가 전통적 상향이 아닌 **다운그레이드** — 가격탄력성의 콘텐츠 측 실증.

## 컬럼 설명

- `tier`(low_end/mid_high/all), `metric`, `value`, `unit`, `year`, `note`, `confidence`

## 출처/수집 메모

- **저가폰 2026 4GB 회귀**(전통적 상향과 역행). mid/high도 최소 표준 근처(업그레이드 둔화).
- 메모리는 폰 BOM의 10~15%. 가격 급등으로 unit cost +8~10%(2025).
- 2026 DRAM 공급성장 16%(역사 20-30% 미만). 메모리 부족으로 출하 -12.9%(1.12B대).

## 메모리 수요 예측 활용

- 콘텐츠 다운그레이드(4GB 회귀)는 **가격탄력성의 콘텐츠 측 실증**(Steam RAM 파일과 동형) — 가격↑→GB/폰↓.
- 모바일 DRAM bit 수요 = 출하(-12.9%) × GB/폰(다운그레이드) = 이중 하방(가격 급등 피드백).
- 단 on-device AI(별도 파일)는 콘텐츠 상향 압력 — 가격이 상쇄.
