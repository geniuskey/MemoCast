# Samsung 파운드리-HBM4 base die 융합·turnkey — extracted notes

> HBM4부터 base die가 로직 공정(파운드리)으로 → 메모리-파운드리 융합. 공급 capacity가 파운드리에 의존.

## 핵심 사실

- **HBM4 base die를 로직 공정으로 제조**: Samsung 4nm / TSMC 3nm·5nm. HBM3E까지는 메모리 부문이 base die 담당.
- Samsung "turnkey"(메모리+파운드리+패키징 일괄): 개발기간 ~20% 단축, 공급망 비용↓. OpenAI 고객 확보 보도.
- **Samsung 평택 파운드리 capacity의 50%+를 자사 HBM4 base die에 배분**(외부 고객 대신).
- SK hynix는 HBM4E 로직 die에 **TSMC 3nm** 검토(Samsung 대비 우위 목적).
- 파운드리 점유 2025: TSMC 69.9%($122.54B), Samsung 7.2%($12.63B).
- 2026 초 Samsung·Micron pin speed 11.7Gbps, stack당 2.8TB/s.

## 메모리 수요 예측 관련성

- HBM4 공급 capacity = min(DRAM fab, **로직 파운드리 base die capacity**, 패키징). 파운드리가 새 게이트.
- Samsung 평택 50%+ HBM 배분은 파운드리 외부 고객 기회비용 = HBM 우선순위 신호.
- 메모리-파운드리 융합은 HBM 공급을 TSMC capacity(CoWoS·로직)에 추가 종속.
