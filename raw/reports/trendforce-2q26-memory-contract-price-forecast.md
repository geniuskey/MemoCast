# TrendForce 2Q26 Memory Contract Price Forecast — extracted notes

> 원천 요약 노트. 수치는 wiki에서 교차검증 후 인용.

## 핵심 수치
- 2Q26 전통 DRAM 계약가 +58~63% QoQ
- 2Q26 NAND Flash 계약가 +70~75% QoQ
- 1Q26 DRAM 계약가 record +90~95% QoQ (직전 분기)
- DRAM 상승률은 둔화, NAND는 직전 ~60%에서 가속
- (모바일) LPDDR4X +70~75% QoQ, LPDDR5X +78~83% QoQ (2Q26)

## 수급/전망
- 북미 CSP의 AI inference 배치 가속 → AI/general-purpose 서버 수요, high-capacity RDIMM이 주 조달 타깃
- CSP가 LTA(장기계약)로 가용 공급 대부분 확보
- enterprise SSD 주문 둔화 조짐 없음, 2026년 명백한 부족 → 의미있는 capa 확장은 2027 말~2028
- 메이커가 server 관련 application으로 capa 재배분 지속
- GDDR: capa 할당 제한으로 공급 제약, 2Q26 추가 가격 상승

## 메모리 수요 예측 관련성
- 분기별 계약가 QoQ % — 가격 시계열 forecast의 직접 입력값
- 1Q26 +90~95% → 2Q26 +58~63% 둔화 곡선 = 가격 모멘텀 변곡 신호
- LTA 기반 공급 잠금이 spot/contract spread 및 수요 가시성에 미치는 영향
