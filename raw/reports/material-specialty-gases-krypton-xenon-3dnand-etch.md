# 희귀가스(크립톤/제논/네온) 메모리 공정 의존 — extracted notes

> 원천 요약 노트. 메모리 공정(특히 3D NAND 딥트렌치 식각)에서 크립톤·제논의 역할과 중국 공급 집중.

## 핵심 수치
- 글로벌 rare gas 시장: 2025년 약 USD 4.1 billion → 2035년 USD 6.8 billion, CAGR 5.2%
- 크립톤/제논은 ASU(극저온 공기분리장치)의 부산물 → 추가 회수·정제 설비 설치 시에만 생산 (구조적 공급 제약)
- 중국은 크립톤·제논·네온 자급 공급 보유, 글로벌 수요 유입 시 타이트화 가능

## 공급/소재 리스크
- 크립톤/제논 + C4F6, CH2F2 식각가스 → 3D NAND 딥트렌치 식각의 핵심 (고단 NAND 적층 증가 시 사용량↑)
- 크립톤: 스퍼터 증착·식각·어닐링·리소그래피에 사용
- 네온: 고정밀 포토리소그래피 광원(엑시머 레이저)의 기반
- 아르곤: 웨이퍼 식각에 광범위 사용
- 중국 공급 집중(Handan Zhengde 등, HBIS Group 기반) → 수출규제·지정학 리스크 노출
- 네온 소싱 변동성·크립톤 가용성이 전자 제조 운영비 상승 요인

## 메모리 수요 예측 관련성
- 3D NAND 적층 단수 증가(200단+ → 300단+)는 딥트렌치 식각 강화 → 크립톤/제논 단위 소비량 증가 → 가스 공급이 NAND bit 증산의 비용·가용성 제약
- DRAM/HBM 식각·증착 공정의 희귀가스 의존 → 공급 쇼크 시 메모리 공급 측 단기 리스크
- 중국 가스 의존도는 메모리 공급망의 China-dependency 항목으로 추적
