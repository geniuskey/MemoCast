# ASML Q1 2026 — Memory EUV Orders Overtake Logic — extracted notes

> ASML 1Q26 실적. DRAM 메이커가 EUV 슬롯 확보 경쟁 → 메모리향 매출이 로직을 처음 추월. EUV는 DRAM 1c node 핵심.

## 핵심 수치
- 메모리가 순 시스템 매출의 51% 차지 (로직 49%) — 사상 처음 메모리 > 로직.
- 분기 출하의 51%가 메모리 생산용.
- 수주 백로그 €38B+ (절반 이상이 EUV).
- EUV 시스템 백로그 45대 (Q1 2026 기록) — TSMC/Samsung Foundry/Intel Foundry 동시 발주(2nm 이하).
- 2026 EUV 출하 목표 60대+ (2025년 48대 대비 증가), High-NA 포함.
- 2027 EUV 출하 약 80대 전망 — 메모리 메이커 수요가 동력.
- High-NA EUV 5~10대 2026년 인도 예정 (대당 약 $370M).

## 공급/장비 영향
- DRAM 1c node로의 공격적 전환이 EUV 채택 가속 → DRAM 비트 생산성·성능 향상.
- 메모리 메이커가 EUV 슬롯 부족으로 경쟁 → 리소그래피가 DRAM 증설의 병목 게이트.
- High-NA 생산능력 연 20대 미만, 백로그 2027년 말까지 → 첨단 노드 capacity 제약.

## 메모리 수요 예측 관련성
- EUV 메모리 비중 51%는 DRAM(특히 HBM용 base die·1c DRAM) 투자 사이클의 강도를 직접 반영.
- 2027 EUV 80대 전망은 2027~2029 DRAM 비트 공급 확장의 선행지표.
