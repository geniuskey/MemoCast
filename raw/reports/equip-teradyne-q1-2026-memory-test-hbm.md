# Teradyne Q1 2026 — Memory Test & HBM4 — extracted notes

> Teradyne는 Advantest와 함께 메모리 ATE 양강. Magnum 7H가 HBM3E/HBM4 테스트. AI가 매출 70%.

## 핵심 수치
- Q1 2026 매출 $1.282B (record, YoY +87%); 비GAAP EPS $2.56.
- 세그먼트: Semiconductor Test $1,111M, Robotics $91M, Product Test $80M.
- AI 관련 수요가 Q1 2026 매출의 70%까지 도달 전망.
- 메모리 테스트 매출 $128M (Q3 2025), QoQ +110%; HBM+DRAM이 메모리 매출의 75%, flash 25%(클라우드 SSD).
- Magnum 7H: HBM3E/HBM4 지원, HBM4E/HBM5 업그레이드 대응; 양산 출하 2025 Q3 시작.

## 공급/장비 영향
- HBM/DRAM이 메모리 테스트 매출의 75% → AI 메모리 테스트 단계가 출하 게이트.
- Magnum 7H 양산 램프는 HBM4 양산 테스트 capacity 확충을 의미.
- 메모리 테스트 시장은 Advantest+Teradyne 합산 45%+ 과점.

## 메모리 수요 예측 관련성
- 메모리 테스트 매출 급증(QoQ +110%)은 HBM/DRAM 비트 출하 증가의 후공정 선행지표.
- AI 매출 70%는 테스트 capex가 AI 메모리 수요에 직접 연동됨을 보여줌.
