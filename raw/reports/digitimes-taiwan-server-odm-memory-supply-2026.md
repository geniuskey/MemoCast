# DigiTimes — 대만 서버 ODM·메모리 수급 (2026) — extracted notes

> 대만 ODM/EMS 매출은 AI 서버 수요(=메모리 수요)의 고빈도 동행 지표.

## 핵심 내용

- 글로벌 메모리 2026년 내내 tight — CSP의 AI 인프라 지출이 DRAM·NAND 공급 성장 초과.
- **AI 서버 수요가 메모리 공급을 2027년까지 lock-up**, 가격 firm 유지 전망.
- 대만 ODM/EMS 2026-03 매출 급증(AI 서버 + 분기말 restocking). Quanta·Wistron·Inventec·Wiwynn·Foxconn·MiTAC이 2026-01 두 자릿수 YoY 성장, Wistron·Wiwynn은 세 자릿수.
- 모듈메이커: Transcend 회장 "공급-수요 압박 2026 내내 심화"(소비 수요 약화에도). ASP 최대 5배 급등 언급.

## 메모리 수요 예측 관련성

- 대만 ODM 월매출(Foxconn 등)은 AI 서버 출하의 nowcasting 프록시 → server DRAM/HBM 수요 동행.
- "2027까지 메모리 lock-up"은 공급 부족 지속 시나리오의 채널 측 확인.
- 모듈메이커 ASP 5배는 채널 가격 전가 강도 지표.
