# GPU 감가상각 논쟁 (Burry thesis) — extracted notes

> GPU 내용연수 가정이 hyperscaler 이익·재투자 여력을 좌우 → AI capex(=메모리 수요) 지속성 변수.

## 핵심 내용

- **Michael Burry(2025-11)**: hyperscaler가 NVIDIA GPU를 5~6년 상각하나 실제 경제수명 2~3년 주장 → 2026-2028 감가상각 **$176B 과소계상**, 이익 과대.
- 예: Microsoft $17B GPU를 6년 상각(3년 대신) → 연 ~$2.9B 이익 과대(2024 순익의 3.3%).
- 현행: hyperscaler GPU 상각 ~6년. 2025 Amazon 일부 서버 내용연수 단축, Meta는 연장(엇갈림).

## 반론

- training 진부화(18~36개월)와 경제 효용(waterfall 모델, 추론 재활용) 구분.
- CoreWeave/분석가: A100이 만료계약서 원가의 95% 유지 → 6년 상각 옹호.

## 메모리 수요 예측 관련성

- 상각 가정이 짧아지면(2~3년) hyperscaler 이익 급감 → capex 둔화 → 가속기/HBM 수요 하방(bear case).
- GPU 교체주기(2~3년 vs 6년)는 가속기 reset 수요(=HBM 재구매) 빈도 → 수요 빈도 변수.
- [[scenario-analysis]] bear case의 회계/재무 트리거(ROI 갭 파일과 연계).
