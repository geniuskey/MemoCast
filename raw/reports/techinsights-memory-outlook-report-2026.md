# TechInsights Memory Outlook Report 2026 — extracted notes

> 원천 요약 노트. 수치는 wiki에서 교차검증 후 인용. (상세 수치는 유료 플랫폼; public 요약 기반 — confidence low)

## 핵심 수치
- AI·datacenter 성장이 record DRAM·NAND 매출 견인; HBM·QLC·hybrid bonding이 가치 증가 주도
- 3D NAND 300단 초과 (wafer bonding, hybrid stacking)
- DRAM은 AI 최적화 HBM 방향으로 이동
- (교차 인용) IDC: 2026 DRAM 공급 +16% / NAND +17% YoY — 역사적 평균 하회

## 수급/전망
- 폭발적 AI workload가 전통 아키텍처 한계 초과 → 공급 제약 심화
- 메모리 기술이 data economy의 bottleneck으로 부상
- 추가 capex가 2026 bit 공급 성장에 미치는 영향 minimal
- MRAM·ReRAM이 IoT·automotive·edge AI에서 확대

## 메모리 수요 예측 관련성
- 기술 로드맵(300단 NAND, hybrid bonding, HBM4) = 장기 bit/원가 trajectory 입력
