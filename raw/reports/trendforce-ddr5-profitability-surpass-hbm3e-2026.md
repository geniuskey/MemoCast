# TrendForce DDR5 vs HBM3e Profitability 2026 — extracted notes

> 원천 요약 노트. 수치는 wiki에서 교차검증 후 인용.

## 핵심 수치
- DDR5 계약가 2026년 내내(특히 1H) 상승 궤도
- 4Q25 DDR5 가격 인상폭 $0.65/Gb 확정
- DDR5 수익성, 2026년 1분기부터 HBM3e 초과 전망
- 1Q26: DDR5 64GB RDIMM이 per-wafer 매출·수익성 모두 HBM 초과
- 가격 격차 축소: HBM3e가 server DDR5 대비 기존 4~5배 → 2026년 말 1~2배로 좁혀짐

## 수급/전망
- 4Q25 server DDR5 계약가가 기대 상회 → wafer 수익성 개선, HBM3e와 수렴 가속
- 높은 DDR5 수익성이 capa crowding 심화 → HBM3e 가격 모멘텀 강화
- 대규모 AI inference workload 전환이 RDIMM 가격 견인

## 메모리 수요 예측 관련성
- DDR5 vs HBM3e 수익성 역전 = 메이커의 capa 배분 의사결정 driver
- per-wafer 수익성 지표가 공급(bit) 배분 → 전통 DRAM 공급 타이트 심화
- 가격 격차(4~5x→1~2x) 수렴은 HBM 프리미엄 정상화 시계열 신호
