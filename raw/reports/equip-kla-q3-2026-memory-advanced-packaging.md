# KLA Q3 FY2026 — Process Control, Memory Mix, AP $1B — extracted notes

> KLA는 검사/계측(process control) 사실상 독점. 첨단패키징(HBM) 매출 2배. 메모리 내 DRAM 84%.

## 핵심 수치
- Q3 FY2026 매출 $3.42B (가이던스 상회), EPS $9.40.
- 차분기 메모리 = Semiconductor Process Control 시스템 매출의 약 18%; 그 안에서 DRAM ~84%, NAND ~16%.
- 첨단 패키징 process control 매출: 2025 $635M → 2026 약 $1B (거의 2배), AI/HBM 수요.
- process control 점유율 2021년 대비 +360bp, 최근접 경쟁사의 약 7배 점유.
- China = FY2025 매출의 약 33%; 수출통제로 CY2026 매출 역풍 $300~350M(Jefferies 추정).

## 공급/장비 영향
- 메모리 process control 내 DRAM 84%는 DRAM(특히 1c·HBM base die) 검사 투자 집중을 의미.
- AP process control $1B는 HBM 스택 검사 수요 급증 → HBM 수율·출하 게이트.
- China 역풍은 레거시 메모리 capex 둔화 신호.

## 메모리 수요 예측 관련성
- HBM 검사/계측 매출 2배는 HBM 양산 capacity 확충의 선행지표.
- DRAM 검사 집중은 DRAM 비트(첨단 노드) 공급 확대를 시사.
