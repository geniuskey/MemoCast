# Collusion Along the Learning Curve: DRAM Cartel — extracted notes

> DRAM 카르텔(1998-2002)의 담합 효과를 learning-by-doing(학습곡선) 관점에서 실증한 산업조직론 논문.

## 문제/목표
- DRAM처럼 learning-by-doing과 다세대 경쟁이 있는 시장에서 담합 효과가 세대별로 어떻게 다른가?
- 학습률이 높은 신세대일수록 담합 이탈 유인이 커 담합 유지가 어렵다는 가설 검증.

## 방법론
- 동적 가격담합 이론모델: 학습률↑ → 이탈 시 미래비용 절감 → 이탈 매력↑ → 담합 약화.
- 실증: 외생적으로 경쟁적인 fringe(비카르텔 공급자)의 점유율 변화로 담합 강도 추정.
- 검정 가설: 담합이 효과적인 제품일수록 fringe 산출이 카르텔 대비 더 확대.

## 핵심 결과
- 카르텔은 학습이 적은 구세대(16Mb)에서 산출을 더 많이 제한.
- 학습이 큰 신세대에서는 담합 유지가 더 어려움 → 세대별 비대칭적 담합.

## 메모리 수요 예측 적용점
- 공급측(가격/산출) 모델링 시 카르텔/과점 행동이 세대별로 다름을 반영해야 함.
- DRAM 가격 형성에서 학습곡선(누적생산→원가하락)을 공급함수에 내장하는 근거.
- 과점 공급자의 산출 조정(감산) 행동을 가격 시나리오에 통합.
