# Real Option Analysis for Capacity Investment Planning for Semiconductor Manufacturing — extracted notes

> 수요 불확실성 하 반도체 capacity 투자(증설)를 real option으로 가치평가한 IEEE 논문.

## 문제/목표
- 반도체 fab은 수십억 달러 투자 + 짧은 제품수명 + 큰 수요 불확실성 → 단순 NPV로 평가 한계.
- capacity 증설 결정에 경영 유연성(option value)을 반영한 평가.

## 방법론
- Real option analysis: capacity 확장 권리를 옵션으로 모델링.
- binomial(이항) 모델 + risk-neutral 방법으로 불확실성 이벤트 트리 전개 및 옵션가치 산정.

## 핵심 결과
- 수요 불확실성이 클수록 "대기/단계적 증설" 옵션의 가치가 커짐.
- real option 평가가 단순 NPV 대비 capacity 투자의 유연성 가치를 더 정확히 측정.

## 메모리 수요 예측 적용점
- 메모리사 CapEx/증설 의사결정의 공급측 모델링: 수요 시나리오별 옵션가치로 증설 타이밍 추정.
- 수요 예측 불확실성(밴드)을 옵션 평가의 변동성 파라미터로 직접 연결.
- "감산/증설 지연" 같은 공급조정 행동을 옵션 행사로 시나리오화.
