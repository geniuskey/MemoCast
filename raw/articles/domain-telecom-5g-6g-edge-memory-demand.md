# 통신 5G/6G·엣지 메모리 수요 — extracted notes

> 기지국·MEC 엣지 서버·라우터가 DDR5 채택을 늘리며 신규 server-class DRAM 수요 형성. 6G·엣지 micro-DC 확산이 중기 동인.

## 핵심 수치
- IT & Telecom 메모리 세그먼트 2034년까지 CAGR 약 **20.6%** 전망.
- 5G 기지국 사이트에 DDR5 탑재 MEC 서버 확산 — Nokia, Ericsson, Huawei 모두 최신 엣지 컴퓨팅 플랫폼에 DDR5 적용.
- Server DRAM은 글로벌 DRAM 내 최고 성장 세그먼트; 2026 Q1 가속 정점, Q2'26 추가 +50~60%+ 가능, 서버 시장 +12.8% 성장 동반.
- 엣지 서버 시장 2035년까지 5G MEC·산업 IoT 채택으로 성장(IndexBox).
- IoT 연결 디바이스 2030년 400억 대 초과 전망.

## 수요 동인
- 5G 대규모 MIMO·실시간 신호처리·네트워크 가상화(NFV/vRAN) → 기지국·엣지 노드 DRAM 함량 상승.
- 6G 대비 metro/suburban micro-data-center 분산 배치 → 저지연 엣지 서버 DRAM 수요.
- AI 기반 네트워크 관리·트래픽 분석이 엣지 추론 메모리 추가 견인.

## 메모리 수요 예측 관련성
- 통신은 server DRAM 수요의 보조 채널(주력은 하이퍼스케일러). 기지국·엣지 자체는 절대 bit 비중 작으나 DDR5/엣지 추론으로 함량 증가.
- 6G 본격화(2030년대)·vRAN 전환이 중기 incremental 수요 옵션.
