# OEM/EMS 메모리 조달·헤지·멀티소싱 (2026) — extracted notes

> 수요측(OEM) 조달 행태가 가수요·재고·allocation 동학을 형성. 수요 신호 왜곡의 미시 기초.

## 핵심 사실

- **멀티소싱·조기 qualify·60~90일 재고 buffer**(현금 여유 시). authorized independent distributor로 allocation 갭 보완.
- **투기적 선주문(speculative order)으로 헤지** — 단 장기 완화 효과 제한(S&P). 공급사가 **계약물량만 공급(allocation-only)**으로 제한 → discretionary 주문 차단.
- S&P Global Mobility: 2026 DRAM 가격 +70~100%(vs 2025). HBM 수요 +70% YoY, DRAM wafer 23% 흡수.
- JIT 탈피·다변화로 resilience.

## 메모리 수요 예측 관련성

- 투기적 선주문·buffer 재고는 **가수요(over-ordering)** 생성 → 단기 수요 과대계상(bullwhip 파일 연계).
- allocation-only 전환은 discretionary 수요를 억제 → 실수요-가수요 구분 중요.
- 60~90일 buffer 빌드업은 재고 사이클(inventory 파일)의 채널 단 증폭.
