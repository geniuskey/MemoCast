# On-device AI 스마트폰 DRAM 견인 — extracted notes

> on-device LLM 실행 요건이 스마트폰 RAM 최소사양을 끌어올림. 모바일 DRAM 콘텐츠 동인.

## 핵심 내용
- **Google Gemini(Intelligence) 12GB RAM 요구**, **Apple Intelligence는 8GB**에서 동작.
- **Galaxy S25/S25+/S25 Ultra 전 모델 12GB RAM** (이전 세대 S25/S25+ 8GB에서 상향, Ultra와 통일).
- S25 Ultra(12GB)도 Gemini Nano v2 구동; 일부 신기능은 cloud 필요.
- 2026년 모바일 local AI sweet spot: 3B~7B 파라미터 모델 + 8GB+ RAM. 7B 모델 4-bit 양자화 시 4~5GB 점유 → 8~12GB 플래그십에서 구동.
- on-device AI가 2026년 niche → 핵심 제품 전략으로 이동.

## 수요 동인
- on-device LLM 상주 메모리 요구 → 플래그십 RAM 8GB→12GB 표준화, 16GB 채택 확대 압력.
- 모델 양자화·NPU 발전이 RAM 요구를 일부 완화하나, 멀티모달/대형 모델은 RAM 상향 견인.

## 메모리 수요 예측 관련성
- 모바일 DRAM bit = 스마트폰 출하량 × 평균 GB/대. on-device AI가 premium mix의 GB/대를 12GB 이상으로 끌어올림.
- Apple 8GB vs Google 12GB 요건 차이 → OEM/세그먼트별 평균 GB 가정에 반영.
- (참고) TrendForce: 2026년 평균 스마트폰 DRAM 약 8.5GB(+10% YoY), premium 12GB 주류.
