# Toshiba Memory sale to Bain-led consortium ($18B) & Kioxia rebrand — extracted notes

> 세계 2위 NAND 업체 Toshiba가 메모리 사업을 Bain 컨소시엄에 약 $18B(2조 엔)에 매각 -> Kioxia 출범. 일본 메모리 자산의 사모펀드 매각 사례.

## 핵심 사실 (연도/기업)
- NAND/플래시 기원: 1980년 Toshiba의 Fujio Masuoka가 플래시 메모리 발명, 1984년 NOR 플래시 발표. Toshiba가 NAND 플래시의 발명사.
- 매각 배경: 미국 원전 자회사 Westinghouse 파산(2016)으로 인한 대규모 손실 -> 재무 구멍 메우기 위해 메모리 사업 매각.
- 거래: 2017-09 발표, Bain Capital Private Equity 주도 컨소시엄에 약 2조 엔(US$18 billion)에 메모리 사업 매각. 2018-06-01 클로징.
- 컨소시엄 구성: Bain Capital(주도), 일본 Hoya, 한국 SK Hynix, 미국 Apple, Kingston Technology, Seagate Technology, Dell Technologies Capital.
- 분사/리브랜드:
  - 2018-06-01 Toshiba Memory Corporation 분사.
  - 2019-03-01 Toshiba Memory Holdings Corporation.
  - 2019-07-18 사명 변경 발표, 2019-10-01 Kioxia Holdings / Kioxia Corporation 공식 출범. ("kioku 기억" + 그리스어 "axia 가치")
- 현재 지분(2025-07-04): Bain Capital 51.11%, Toshiba 30.50%, Hoya 3.00%.
- 상장: 2024-12 도쿄 IPO 상장(별도 보도, 약 $800M 규모).
- WD/SanDisk와의 관계: WD/SanDisk가 Kioxia 설비를 통해 플래시 공동 생산(요카이치/기타카미 합작 JV).

## 구조 변화 / 교훈
- NAND 발명사인 Toshiba조차 본업 외 손실로 메모리 사업을 사모펀드에 넘긴 사례 -> 메모리는 막대한 자본 투입과 재무 체력이 필수.
- 사모펀드 인수 후 Kioxia는 한동안 IPO 지연(시황 악화) -> NAND 산업의 강한 사이클성을 방증.
- NAND는 Kioxia-WD/SanDisk JV가 한 축, Samsung/SK hynix/Micron이 다른 축 -> DRAM보다 분산된 과점.

## 메모리 수요 예측 관련성
- Toshiba/Kioxia 사례는 NAND 공급자의 재무 취약성이 투자(공급)에 직결됨을 보여줌 -> 공급 측 변수로서 업체 재무·지배구조 모니터링 필요.
- Kioxia IPO 타이밍(시황 의존)은 NAND 사이클 판단의 보조 신호로 활용 가능.
