# 네트워킹(스위치/라우터) DDR 메모리 수요 (2026) — extracted notes

> AI 데이터센터 네트워킹(400G/800G 스위치)이 DDR5 수요 신규 축. HBM 네트워킹(별도 domain-networking-switch-hbm)과 구분.

## 핵심 사실

- AI 데이터센터용 **400G/800G 스위치** 등장 → 전통 DRAM 한계, DDR5 모듈 업그레이드 필수.
- **라우터 메모리 비용 ~7배 상승**(DRAM 급등 전이).
- DRAM 가격 YoY +171%, AI 데이터센터가 고급 DRAM ~70% 소비.
- 통신사·클라우드에 DRAM이 더 이상 passive BOM 아님 — 조달이 배포 일정·refresh·TCO 직접 좌우.
- HPE: DRAM/NAND 원가 상승 2026 지속, 시장 전가.

## 메모리 수요 예측 관련성

- 네트워킹 장비(스위치/라우터)는 server 외 데이터센터 DRAM 수요의 보조 축(buffer/제어 메모리).
- 400G/800G 전환이 장비당 DDR 콘텐츠 상향 → AI 인프라 DRAM 수요에 가산.
- 라우터 메모리 7배 상승은 가격 전가가 네트워킹 장비까지 확산한 증거.
