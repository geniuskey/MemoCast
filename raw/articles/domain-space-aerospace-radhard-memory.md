# 우주/항공 rad-hard 메모리 — extracted notes

> 위성·메가컨스텔레이션·우주 탐사용 방사선 경화(rad-hard) 메모리. 고신뢰·고단가 niche, bit 절대량은 작지만 ASP 매우 높음.

## 핵심 수치
- 글로벌 **rad-hard 메모리 시장 2024년 $1.27B → 2033년 $2.68B, CAGR 8.5%** (Dataintelo).
- rad-hard 전자부품 시장 2025년 약 $2.0B → 2032년 $3.5~3.8B, CAGR 6~8% (Electronicsmedia / MarketsandMarkets는 2030년 $2.30B 인용).
- 우주(space) 애플리케이션이 2024년 약 **50% 점유** — 위성·프로브·로버.
- 2023년 200대 이상 위성 발사, 메가컨스텔레이션 기하급수 성장 전망.
- 메모리 타입: SRAM, DRAM, Flash로 세분. 메모리 컴포넌트 세그먼트가 2024~2030 최고 CAGR 전망(센서·프로세서 데이터 처리 증가).
- 주요 기업: Microchip, BAE Systems, Renesas, Infineon, STMicroelectronics.

## 수요 동인
- LEO 메가컨스텔레이션(Starlink 등)·소형 위성 양산화 → 컴팩트·저전력·고신뢰 메모리 수요 증가.
- 민간 우주(NewSpace) 참여 확대, 위성당 센서·온보드 처리량 증가.

## 메모리 수요 예측 관련성
- bit 절대량은 상용 대비 극소수이나, 단가(ASP)·마진이 매우 높아 specialty 메모리 매출 기여.
- 위성 발사 수 증가가 unit 동인. 표준 상용 메모리 bit pool에는 영향 미미하나 specialty NAND/DRAM/SRAM 수요로 분류.
