# 게이밍/콘솔 GDDR 메모리 — extracted notes

> 콘솔·GPU의 GDDR 탑재량과 AI/자동차의 GDDR 잠식. 그래픽 메모리 수요 동인. (PS6 사양은 루머, confidence low.)

## 핵심 내용
- **PS5**: 공식 16GB GDDR6, 대역폭 448 GB/s.
- **PS6 (루머)**: 30GB GDDR7, 대역폭 640 GB/s (PS5 대비 +42%). 메모리 가격 급등으로 출시 2028년 이후로 연기 가능성.
- GDDR7로 전환했지만 **GDDR6가 여전히 물량의 대부분** (PS5/PS5 Pro, 다수 Radeon·구형 GeForce, Tesla 차량 플랫폼).
- Tesla가 Samsung에 GDDR6 공급 확대 요청 → PS5·게이밍 GPU용 DRAM 부족 우려.
- GDDR7는 2026~2027년 midrange GPU까지 확산 전망.
- 표준 DRAM contract 가격 2Q26 +58~63% QoQ → GPU·콘솔 메모리 원가 압박.

## 수요 동인
- 차세대 콘솔(PS6 30GB)·GPU의 GDDR 용량 증가.
- AI 데이터센터·자동차(Tesla)가 GDDR6/7 수요를 잠식 → 게이밍 segment와 capacity 경쟁.
- TrendForce: GDDR7 1GB = 표준 DRAM 약 1.7배 wafer 소모.

## 메모리 수요 예측 관련성
- 콘솔당 GDDR 용량(16GB→30GB)이 세대 전환 시 약 2배 → 콘솔 출하 × GB로 환산.
- GDDR가 표준 DRAM 대비 1.7배 capacity 소모 → AI 외 그래픽 수요도 공급 잠식 요인.
- PS6 출시 시점 불확실(2028+) → 콘솔 GDDR 수요 변곡점 지연 리스크.
