# 차량당 메모리 콘텐츠 (Bosch/Continental Tier-1) 2025 — extracted notes

> 자동차 OEM/Tier-1의 차량당 DRAM/NAND 콘텐츠·자율주행 단계별 요구량 원천 노트.

## 핵심 수치
- 차량당 메모리(2021 기준): DRAM 약 5.3GB + NAND 약 50GB
- 하이브리드(HEV) 듀얼 파워트레인 관리: DRAM 4-6GB (Bosch/Continental 공급망)
- L2 자율주행 센서 퓨전: 전용 DRAM 2-4GB
- 자동차 데이터로거 NAND: 32-128GB 밴드가 2025 점유 31.74%; L3 개발 프로그램은 512GB+ 테스트(CAGR 8.15%, 최고 성장)
- DDR5: 2025 자동차 메모리 점유 18.7%, CAGR 21.3%(~2034)
- DDR5 채택 프리미엄 플랫폼: BMW iX M60, Mercedes-Benz EQS, Porsche Taycan
- Micron 전망: 2030년 자율차 1대당 NAND 4TB; 하이엔드 차량 DRAM+NAND 합산 2TB 도달 가능

## 수요 동인 (OEM별)
- ADAS/자율주행 레벨 상승 = 차량당 DRAM/NAND 콘텐츠 급증 (L2 → L3 전환이 핵심 변곡)
- Tier-1(Bosch, Continental)이 도메인 컨트롤러/ADAS 모듈에 메모리 직접 소싱
- 인포테인먼트 + 자율주행 + 커넥티비티 동시 실행 → DDR5 전환 가속

## 메모리 수요 예측 관련성
- 자동차 단위 출하는 완만하나 차량당 콘텐츠(GB) 급증이 비트 수요 동인 (스마트폰과 유사 구조)
- 차량 메모리는 장수명·고신뢰(AEC-Q) 요구 → 별도 공급 캐파 필요
- L3+ 보급 시 차량당 TB급 메모리 → 자동차가 의미있는 비트 수요 풀로 부상
