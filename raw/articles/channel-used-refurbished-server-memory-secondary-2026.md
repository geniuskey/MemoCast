# 중고/리퍼 서버 메모리 secondary 시장 (2026) — extracted notes

> 가격 급등기에 secondary 공급이 수요의 안전판이자 가격 신호.

## 핵심 수치

- DDR5 kit 가격 12개월 +478%, 신규 서버 구성에 unit당 $10,000~27,000 프리미엄(ITAD DDR4 대안 대비).
- 폐기(decommissioned) 14G/15G 플릿 DDR4 모듈이 2025-12 대비 **+30~50%**.
- Samsung·SK가 DDR4 라인 축소 → 2027 전 추세 반전 어려움.
- ITAD 채널: Fortune 500·hyperscaler·정부가 4~5년 refresh 후 청산 → 2021-2023 세대 HW 공급.

## 수요/공급 신호

- 신품 가격 급등으로 중고 DDR4 수요 급증(유지보수·업그레이드·refresh). secondary 공급이 일부 수요 흡수.
- "버려질 RAM이 $20,000 가치"(TechSpot) — secondary 가치 재평가 사례.

## 메모리 수요 예측 관련성

- secondary 시장은 신품 수요의 대체재 → 가격 급등 시 신품 수요를 일부 상쇄(수요 탄력성 요인).
- ITAD 공급(refresh 사이클)은 예측 가능한 secondary 공급 흐름 → 유효 공급에 가산.
- 레거시(DDR4) 단종 + secondary 프리미엄은 세대전환 지연 신호.
