# 신흥국 스마트폰 메모리 수요 (2026) — extracted notes

> 메모리 가격 급등이 entry/mid 세그먼트(신흥국 backbone)를 직격 → 물량 위축.

## 핵심 수치

- **동남아 1Q26 -9% YoY**(21.6M대), ASP 사상 최고 $349(+19% YoY, 메모리 인플레).
- **인도 1Q26 -5%**(30.9M대). entry-level -59% YoY(점유 18%→8%), 2Q26 entry 가격 +18~20%.
- **중동·아프리카 -20.6% YoY** 전망, APAC(중·일 제외) -13.1%.
- entry/mid 세그먼트가 DRAM/NAND 원가 비중 높아 가장 큰 가격 인상 흡수. <$100 폰 "permanently uneconomical" 가능.

## 메모리 수요 예측 관련성

- 메모리 가격→entry/mid 폰 가격→신흥국 물량 위축의 탄력성 피드백(가격 전가, oem-cloud 파일 연계).
- 신흥국은 unit 물량 backbone이나 디바이스당 메모리 콘텐츠 낮음 → bit 수요 영향은 ASP↑ vs unit↓ 상쇄.
- 세그먼트별(entry/mid/premium) 차등 탄력성 → structural model 세분화.
