# 서버 ODM GB200 NVL72 출하 & HBM 콘텐츠 — extracted notes

> NVIDIA GB200 NVL72 랙 출하량과 ODM 공급망, 랙당 HBM 콘텐츠 원천 노트.

## 핵심 수치
- GB200 NVL72: 2025년 4월 단월 1,500 units 출하 (Q1 2025 전체 1,000 units 대비 급증)
- GB200 NVL72 랙 구성: 72 x Blackwell B200 GPU, 랙당 HBM3e 약 130TB pool
- Blackwell가 2025년 NVIDIA 하이엔드 GPU 출하의 80%+ 차지 (TrendForce)
- 주요 ODM: Foxconn(Hon Hai), Quanta, Wiwynn, Supermicro
  - Quanta: Meta/AWS/Google 장기 파트너, GB200/GB300 랙 확대 + Oracle 신규 수주
  - Wiwynn: Meta/Microsoft 협업 심화, ASIC 기반 모델 중심, AWS Trainium 핵심 공급사
  - Supermicro: AI 서버가 올해 주 성장 동력, 다수 GB200 랙 프로젝트 확보
  - Foxconn: Oracle 북미 AI DC 확장으로 수주 증가

## 수요 동인 (OEM별)
- AI 서버 랙당 HBM 콘텐츠가 압도적(130TB/rack) → HBM 비트 수요의 핵심 견인
- ODM별 클라우드 고객 매핑: Quanta↔Meta/AWS/Google/Oracle, Wiwynn↔Meta/MS/AWS(Trainium)
- NVIDIA가 ODM에 부품 조달 권한 부여 → ODM이 메모리 직접 구매 주체로 부상

## 메모리 수요 예측 관련성
- 랙당 HBM 130TB는 일반 서버 대비 수백 배 메모리 콘텐츠 → AI 서버 출하 증가가 HBM 수요 폭증 직결
- GB200 NVL72 출하 ramp(Q1 1,000 → 4월 1,500/월)은 HBM/서버 DDR5 비트 수요 선행 지표
- ASIC 서버(Trainium 등)도 HBM 소비 → NVIDIA 외 수요 추가
