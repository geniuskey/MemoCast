# AI 서버 수요가 2Q26 메모리 가격 견인 — extracted notes

> server DRAM 우선 배분과 CSP 장기계약(LTA)이 2026년 메모리 가격/수요 구조를 규정.

## 핵심 내용
- 2Q26 contract 가격 전망 (QoQ):
  - **Conventional(표준) DRAM: +58~63%**.
  - **NAND Flash: +70~75%**.
- 공급자들은 수익성이 높은 **server DRAM 우선 배분**, 주요 고객과 **장기계약(LTA)** 체결로 향후 capacity 확장 뒷받침.
- 북미 CSP들이 AI inference 롤아웃 가속 → server memory 수요 견인.
- **High-capacity RDIMM이 주요 조달 대상(primary procurement target)**, 타이트한 재고 속 안정 공급 위해 CSP가 장기계약에 적극적.
- 공급자들은 제품군 간 가격 격차 축소를 위한 catch-up pricing 진행, capacity를 HBM·server로 재배분하며 소비자 segment 압박.

## 수요 동인
- AI inference 확산 → 고용량 RDIMM(서버 DRAM) 수요 급증.
- HBM·server 우선 배분 → 소비자용 DRAM/NAND 공급 위축 → 전 제품군 가격 동반 상승.

## 메모리 수요 예측 관련성
- CSP의 LTA는 수요 가시성을 높여 capacity 투자/가격을 사전 고정 → 예측 모델에서 "계약 기반 수요" 비중 확대.
- 표준 DRAM +58~63%, NAND +70~75% QoQ는 2026년 가격 사이클 강도를 보여주는 핵심 지표.
