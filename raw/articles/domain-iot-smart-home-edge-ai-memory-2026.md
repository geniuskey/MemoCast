# IoT/스마트홈/엣지 AI 메모리 수요 (2026) — extracted notes

> IoT는 디바이스당 메모리 작으나 대수 방대. 엣지 AI 침투가 콘텐츠 상향 변곡.

## 핵심 사실

- IoT 수요 안정적이나 폭발적 아님 — 엣지 OEM이 부품가 변동성에 신규 출시 지연.
- DRAM fab이 HBM/고성능 server 우선 → 레거시 DRAM·entry NAND(노트북/SSD/IoT) output 감소(공급 제약).
- IoT Analytics: **2026이 IoT 엣지 AI 확산 변곡점** — AI 칩셋이 센서·연결모듈·산업PC·중급 게이트웨이로 확대.
- 메모리 부족으로 RISC-V IoT 등 저비용 대안 부상(iottechnews).

## 메모리 수요 예측 관련성

- IoT는 대수 방대·디바이스당 메모리 소형 → 엣지 AI 침투(콘텐츠↑)가 bit 수요 변곡 변수.
- 레거시 메모리 공급 제약이 IoT 부품 가용성 직격(crowding out의 long-tail 전이).
- 엣지 AI 칩셋 출하(LPDDR/저전력 NAND)가 신규 수요 축(domain-edge·edge-ai 파일 연계).
