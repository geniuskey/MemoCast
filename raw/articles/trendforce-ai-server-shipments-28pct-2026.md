# 글로벌 AI 서버 출하 +28% YoY (2026) — extracted notes

> AI 서버 출하량 = HBM/server DRAM bit 수요의 1차 곱셈 인자. 가속기 mix(GPU vs ASIC) 포함.

## 핵심 내용
- 2026년 글로벌 AI 서버 출하 **+28% YoY 초과** (북미 CSP, sovereign cloud, 자체 ASIC, edge AI inference 견인).
- 가속기 구성:
  - **GPU 약 69.7%** (여전히 주류).
  - **ASIC 약 27.8%** (2023년 이래 최고; Google·Meta 자체 ASIC 확대). ASIC 서버 출하 성장률이 GPU 기반을 상회 전망.
- 전체 서버(일반 포함) 출하: 2026년 **+12.8% YoY** (AI 서버가 주 성장 동력).
- (별도 보고서) 일반 서버는 부품 lead time 장기화로 2026년 **+13% YoY** 수준.

## 수요 동인
- CSP의 AI capex 지속 + sovereign cloud(정부 주도) 신규 수요.
- ASIC 비중 상승 → HBM/GDDR/server DRAM 조합 다변화(ASIC도 HBM 다수 탑재).

## 메모리 수요 예측 관련성
- AI 서버 +28%는 HBM bit 수요(+70%)와 가속기당 용량 증가를 곱해 메모리 수요로 환산하는 출발점.
- GPU 69.7% / ASIC 27.8% mix는 가속기별 HBM 용량 가정에 직접 반영.
- 일반 서버 +12.8~13%는 표준 server DRAM(RDIMM) 수요 베이스라인.
