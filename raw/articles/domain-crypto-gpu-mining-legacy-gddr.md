# 크립토 마이닝 레거시 GDDR 수요 — extracted notes

> GPU 마이닝은 GDDR6/GDDR6X 메모리를 소비하는 축소 niche. ASIC 지배·전기료 상승으로 신규 수요 미미, 레거시 카드 재활용 중심.

## 핵심 수치
- 2026년 마이닝 GPU 표준 메모리: **GDDR6 / GDDR6X**.
- RTX 4090: **24GB GDDR6X** — 고급 해시레이트.
- 메모리 하드 알고리즘(Autolykos/Ergo 등): VRAM **≥8GB, 대역폭 ≥448 GB/s** 요구.
- 6GB GDDR6 카드는 레거시 알고리즘 처리 가능.
- 수익성 조건: 전기료 ≤ $0.07/kWh, 메모리 하드 코인 타깃, 중고 RTX 30-series 보유 시에만 viable.

## 수요 동인
- 대형 체인 ASIC 지배·ETH PoS 전환 이후 GPU 마이닝은 hobbyist/저전력 지역 중심으로 축소.
- 메모리 하드 코인(Ravencoin, Ergo)이 고대역폭 GDDR을 요구하는 잔존 수요원.

## 메모리 수요 예측 관련성
- 신규 GDDR bit 수요 기여는 미미·감소 추세. 마이닝은 더 이상 메모리 수요의 swing factor 아님(과거 2017/2021 GDDR 사이클과 대비).
- 게이밍 GPU GDDR 수요에 부수적·잡음 수준. 수요 예측에서 무시 가능 수준의 niche로 분류.
