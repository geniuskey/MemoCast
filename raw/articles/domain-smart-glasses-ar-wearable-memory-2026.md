# 스마트글래스/AR 웨어러블 메모리 (2026) — extracted notes

> 신흥 폼팩터. 디바이스당 메모리는 작으나 출하 급증 시 LPDDR/저전력 NAND 수요 증분.

## 핵심 수치

- **Ray-Ban Meta Display**: Qualcomm Snapdragon AR1 Gen1, **32GB 저장, 2GB RAM**. LLAMA 4 통합.
- Gen2 $299, Display(EMG 손목밴드) $799. Gen3 2026(지연 시 2027), screenless, 배터리 6~8h.
- on-device AI(Meta AI) 음성 어시스턴트.

## 메모리 수요 예측 관련성

- 디바이스당 메모리(2GB RAM/32GB NAND)는 스마트폰 대비 작으나, 출하 대수 급증이 누적 LPDDR/저전력 NAND 수요.
- 온디바이스 AI 강화 시 글래스 RAM/저장 콘텐츠 상향 가능(차세대).
- 웨어러블은 long-tail 수요처 — base 시나리오의 점진 증분(절대 bit는 소형).
