# 클라우드 게이밍/게임 스트리밍 메모리 수요 (2026) — extracted notes

> 게이밍이 로컬(GDDR)에서 클라우드(서버 GPU)로 이동 — 메모리 수요 위치 전이.

## 핵심 수치

- 클라우드 게이밍 시장 $8B(2025)→$25B+(2030, Grand View). GeForce Now 25M 유저(2025), xCloud(Game Pass Ultimate 34M).
- NVIDIA GeForce Now "Ultimate"=클라우드 RTX 5090(4K 240fps). 동남아·남미 데이터센터 확장.
- Grace Hopper·MI300X로 GPU 호스팅 비용 -40%(2023 대비).
- **Xbox Helix 가격이 RAM 위기 영향**(2026-04) — 클라우드 컴포넌트 중요성↑.

## 메모리 수요 예측 관련성

- 클라우드 게이밍은 게임 메모리를 로컬(소비자 GDDR/DRAM)→데이터센터 서버 GPU로 전이.
- RAM 위기가 클라우드 게이밍 가격에 영향(메모리 가격이 서비스 비용 전가).
- 소비자 GPU 가격 급등(gddr7 파일)이 클라우드 게이밍 채택 가속 가능 → 수요 위치 이동(net bit 영향 양면).
